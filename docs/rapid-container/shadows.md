# Shadow

Apply shadow to your container

## How to apply a shadow

Shadows are applied with the following pattern `shadow-[range]`.

Here is an example

```dart
RapidContainer(
  styles: "shadow-md",
  child: MyWidget()
);
```
## Accepted shadow symbols

* `shadow-sm`
* `shadow-md`
* `shadow-lg`
* `shadow-xl`
* `shadow-2xl`
* `shadow-none`
