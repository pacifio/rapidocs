# Size

Set container width or height with rapid

## How to apply size

You can apply width with the following pattern `w-[range]` or `h-[range]`.

## Size example

```dart
RapidContainer(
  styles: "w-10",
  child: MyWidget()
);

RapidContainer(
  styles: "h-8",
  child: MyWidget()
);

RapidContainer(
  styles: "w-10 h-10 rounded-full bg-grey-500", // full circle
  child: MyWidget()
);
```
