# Border radius

Apply a border radius to your container

## How to apply a shadow

Border radius is applied with the following pattern `rounded-[range]`

Here is an example

```dart
RapidContainer(
  styles: "rounded-xl",
  child: MyWidget()
);

RapidContainer(
  styles: "rounded-full", // completely round
  child: MyWidget()
);
```
## Accepted border radius symbols

* `rounded-none`
* `rounded-xs`
* `rounded-sm`
* `rounded-md`
* `rounded-lg`
* `rounded-xl`
* `rounded-2xl`
* `rounded-3xl`
* `rounded-full`
