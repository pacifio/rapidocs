# Text size

Set your text size with rapid

## How to apply a size to text

Sizes are applied with the following symbol pattern `text-[range]`

Here is an example

```dart
RapidText(
  text: "Demo",
  styles: "text-xs",
);

RapidText(
  text: "Demo",
  styles: "text-2xl",
);
```

## Accepted font weights

* `text-xs`
* `text-sm`
* `text-base`
* `text-lg`
* `text-xl`
* `text-2xl`
* `text-3xl`
* `text-4xl`
* `text-5xl`
* `text-6xl`
* `text-7xl`
* `text-8xl`
* `text-9xl`
