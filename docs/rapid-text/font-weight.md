# Font weight

Set your font weight with rapid

## How to apply a font weight to text

Font weights are applied with the following symbol pattern `font-[range]`

Here is an example

```dart
RapidText(
  text: "Demo",
  styles: "font-thin",
);

RapidText(
  text: "Demo",
  styles: "font-bold",
);
```

## Accepted font weights

* `font-thin`
* `font-extralight`
* `font-light`
* `font-normal`
* `font-medium`
* `font-semibold`
* `font-bold`
* `font-extrabold`
* `font-black`
